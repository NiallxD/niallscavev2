Bookshelf

I was never into reading when I was younger, and throughout my twenties I spent that much time studying that I really couldn't face reading even more words on a page. Since finishing my masters degree I've been able to put more time towards reading, and I love it!

Here are the books I've read and recommend. Some books don't make it into my bookshelf if I don't feel they are recommendable. You can view my highlights by clicking the 📝 link below each book.

TABLE WITHOUT ID
  "<img src='" + header-image + "' style='height:300px;'/>" as Cover,
  "📖 " + title as "Title",
  "👤 " + author as "Author",
  "👀 " + read as "Read",
  "📝 " + highlights as "Highlights",
  goodreads as "GoodReads"
WHERE Type = "Bookshelf"
SORT read desc
