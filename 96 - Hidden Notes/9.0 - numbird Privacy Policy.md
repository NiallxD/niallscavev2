# numbird Privacy Policy

# Privacy Policy

_Last updated: March 2026_

numbird is developed by Niall Bell. This policy explains what data the app handles, how it is used, and your rights.

If you wish to get in touch with questions or comments, please email Niall at numbird@niallbell.co.uk. I'm always open to feedback and comments.

---

## What Data Is Used

**📱 Observation Data**
Your eBird data (imported as a CSV/ZIP file) is stored entirely on your device. It is never uploaded to any numbird server. numbird has no servers.

**🔑 eBird API Key**
numbird uses a default eBird API Key. You can choose to use your own API key if you wish. If you choose to add your own API Key it will be stored locally in your device's UserDefaults. It is sent only to eBird's servers when making API requests on your behalf (e.g. nearby sightings, target species). It is not stored anywhere else.

**📍 Location Data**
When you use live features which have a geographic component, such as Nearby Sightings or Target Species, you can provide Country, Region, and County data, which is sent directly to the eBird/GBIF API to return geographically relevant results. numbird itself cannot access your device location (via GPS or otherwise). The app includes a feature which allows you to set a 'home location' which allows the app to pre-zoom to your 'home location'. You can set this to be anywhere, and it only changes when you change it. The location is set by dropping a pin on the map and doesn't use any location services (GPS etc).

---

## What We Don't Do

**🚫 No Analytics**
numbird contains no analytics SDKs, crash reporting services, or usage tracking of any kind. The only time I receive information from you is if you email me, or if you report issues using Apple's own feedback functions provided in the App Store or TestFlight apps.

**📢 No Advertising**
numbird contains no advertising networks or ad-tracking SDKs.

**👤 No Account Required**
numbird does not require you to create an account. No personal information (name, email, etc.) is collected by numbird.

**💰 No In-App Purchases**
numbird does not use any in-app purchases. There is a link to 'Buy Me A Coffee' for optional donations.

---

## Third-Party Services

**🐦 eBird (Cornell Lab of Ornithology)**
Live features connect directly to the eBird API. Your API key and location coordinates are sent to eBird's servers. eBird's own privacy policy applies to data they receive. See ebird.org for details.

**🌎 GBIF (Global Biodiversity Information Faciltiy**
Live features connect directly to the GBIF API. This API is keyless and we only send the location you have selected to explore. GBIF's own privacy policy applies to data they receive. See ebird.org for details.

---

## Data Deletion

**🗑️ Deleting Your Data**
You can delete all locally stored observation data at any time from the Dashboard. Uninstalling numbird removes all app data from your device.
