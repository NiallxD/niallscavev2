[!HOT] My Blog Has Moved Over the past year I've found writing this blog to be a little bit robotic. I decided to migrate to a form which is more natural to me. The posts below won't go anywhere, and I may add to this from time to time, but from now on I'm writing over here

TABLE WITHOUT ID
  "<img src='" + header-image + "' style='height:160px;'/>" as Cover,
  file.link as title,
  "📅 " + created as Created,
  "🔄 " + date(updated) as Updated,
  "💭 " + Type as Type,
  join(file.tags, " ") as Tags
WHERE Type = "Blog Post" OR Type = "Photo Story" OR Type = "Photo Analysis" OR Type = "Photo Feed"
SORT date(updated) DESC
