numbird Privacy Policy

Privacy policy for the numbird iOS app.

Last updated: March 2026

numbird is developed by Niall Bell. This policy explains what data the app handles, how it is used, and your rights.

If you wish to get in touch with questions or comments, please email numbird@niallbell.co.uk. I’m always open to feedback.

What Data Is Used

📱 Observation Data Your eBird data (imported as a CSV/ZIP file) is stored entirely on your device. It is never uploaded to any numbird server. numbird has no servers.

🔑 eBird API Key numbird uses a default eBird API Key. You can choose to use your own. If you add your own API key it is stored locally in your device’s UserDefaults and is sent only to eBird’s servers when making API requests on your behalf (e.g. nearby sightings, target species). It is not stored anywhere else.

📍 Location Data When you use live features with a geographic component (such as Nearby Sightings or Target Species), you can provide Country, Region, and County data, which is sent directly to the eBird/GBIF API to return relevant results. numbird itself cannot access your device location via GPS. The app includes a ‘home location’ feature which pre-zooms to a location you set by dropping a pin — it does not use location services.

What We Don’t Do

🚫 No Analytics numbird contains no analytics SDKs, crash reporting services, or usage tracking of any kind. The only time I receive information from you is if you email me, or report issues via Apple’s own feedback functions in the App Store or TestFlight.

📢 No Advertising numbird contains no advertising networks or ad-tracking SDKs.

👤 No Account Required numbird does not require you to create an account. No personal information (name, email, etc.) is collected by numbird.

💰 No In-App Purchases numbird does not use any in-app purchases. There is a link to ‘Buy Me A Coffee’ for optional donations.

Third-Party Services

🐦 eBird (Cornell Lab of Ornithology) Live features connect directly to the eBird API. Your API key and selected location coordinates are sent to eBird’s servers. eBird’s own privacy policy applies to data they receive. See ebird.org for details.

🌎 GBIF (Global Biodiversity Information Facility) Live features connect directly to the GBIF API. This API is keyless and only the location you have selected to explore is sent. GBIF’s own privacy policy applies to data they receive.

Data Deletion

🗑️ Deleting Your Data You can delete all locally stored observation data at any time from the Dashboard. Uninstalling numbird removes all app data from your device.

Contact

✉️ Questions? Contact niall@niallbell.co.uk with any questions about this privacy policy.

This policy applies to the numbird iOS app only.
